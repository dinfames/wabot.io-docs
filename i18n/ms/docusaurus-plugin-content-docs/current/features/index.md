# Ciri-ciri Utama

Wabot menggabungkan tujuh keupayaan berkuasa dalam satu papan pemuka bersatu. Setiap ciri adalah bebas tetapi berintegrasi dengan lancar — anda boleh merantai Chatbot ke dalam Automasi, menghalakan dari Balasan Automatik ke Sembang Langsung, dan menyiarkan ke segmen yang dicipta dalam Audiens.

## Tujuh Tiang Utama

- **[Chatbot](/docs/features/chatbots)** — Bot berkuasa AI yang menjawab FAQ, menapis prospek, dan menutup jualan 24/7.
- **[Siaran](/docs/features/broadcast)** — Hantar kempen pukal kepada kumpulan kenalan atau segmen dengan penjadualan.
- **[Automasi](/docs/features/automation)** — Pembina aliran kerja visual yang dicetuskan oleh webhook atau label WhatsApp.
- **[Balasan Automatik](/docs/features/autoresponder)** — Balasan automatik berdasarkan kata kunci untuk salam, FAQ, dan sokongan pantas.
- **[Sembang Langsung](/docs/features/live-chat)** — Inbox pasukan dikongsi dengan penapis: Semua, Belum Dibaca, Widget, Chatbot Aktif/Tidak Aktif, Diarkib.
- **[Templat](/docs/features/templates)** — Komponen mesej boleh guna semula: pra-lulus, senarai, butang, undian, urutan, balasan pantas.

## Memilih Ciri yang Betul

| Saya mahu... | Gunakan ciri ini |
|--------------|------------------|
| Hantar promosi kepada 500 kenalan | **Siaran** |
| Balas automatik kepada kata kunci "harga" | **Balasan Automatik** |
| Guna AI untuk jawab soalan harga | **Chatbot** |
| Cetuskan mesej apabila borang dihantar | **Automasi** (pencetus webhook) |
| Susulan apabila label WhatsApp berubah | **Automasi** (pencetus label) |
| Biar pasukan saya balas pelanggan bersama | **Sembang Langsung** |
| Guna semula templat mesej WABA pra-lulus | **Templat → Pra-Lulus** |
| Simpan balasan tersimpan untuk balasan pantas | **Templat → Balasan Pantas** |
