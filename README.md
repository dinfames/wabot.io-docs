# Wabot.io Documentation

Bilingual (English + Bahasa Melayu) documentation for [Wabot.io](https://wabot.io) — the all-in-one WhatsApp automation platform.

Built with [Docusaurus](https://docusaurus.io/).

## Local Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.

**Live site:** [https://dinfames.github.io/wabot.io-docs/](https://dinfames.github.io/wabot.io-docs/)
